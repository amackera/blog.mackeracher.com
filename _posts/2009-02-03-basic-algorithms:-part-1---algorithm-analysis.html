Today's topic will be...

Algorithm Analysis:

Given two algorithms, how can we tell which one is superior? One way would be to implement both and run them on the same hardware---but that is expensive and time consuming. It would be nice if we could analyze an algorithm mathematically to determine its properties. Not only will we be able to select between two competing algorithms, but having a technique will help answer any lingering questions about whether we have implemented an algorithm correctly and caught all the bugs. Come to the rescue is Professor Knuth's technique of asymptotic algorithm analysis.
